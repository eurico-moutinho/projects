## Getting Started
### Prerequisites
- Python 3.13
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

### Install the required packages
1. pip install pandas
2. pip install matplotlib
3. pip install seaborn
4. pip install jupyter
